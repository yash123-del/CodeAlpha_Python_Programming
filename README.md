# CodeAlpha Python Programming Tasks

This repository contains my projects completed during the Python Programming Internship at **CodeAlpha**. It features three distinct applications focusing on game logic, data management, and system automation.

## 🚀 Projects Overview

### 1. Hangman Game (`hangman.py`)
A classic word-guessing game where the user tries to figure out a hidden word by suggesting letters within a limited number of attempts.
*   **Key Features:** Random word selection, visual hangman stages, and win/loss tracking.

### 2. Stock Portfolio Tracker (`portfolio_tracker.py`)
A tool to help users manage their stock investments. It allows users to add, remove, and track the performance of various stocks.
*   **Key Features:** Real-time data simulation (or API integration), profit/loss calculation, and portfolio diversification overview.

### 3. Task Automation (`automation_script.py`)
A Python script designed to automate repetitive file management tasks to improve productivity.
*   **Key Features:** Automatic file sorting by extension, directory cleanup, and log generation.

## 🛠️ Tech Stack
*   **Language:** Python 3.x
*   **Libraries used:** `random`, `os`, `json` (and any others like `requests` or `pandas` if used).
